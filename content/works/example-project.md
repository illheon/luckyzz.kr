# Example Project

This is an example project showcasing minimal design principles and clean typography.

## Overview

A portfolio website built with modern web technologies, featuring:
- Minimal, monochrome design
- Custom typography system
- Fast performance
- Responsive layout

## Technologies

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: PP Lettra Mono, Arita Buri, Mulche

## Design Philosophy

The design follows these principles:
1. **Minimal** - Remove everything unnecessary
2. **Monochrome** - Focus on content, not colors
3. **Typography** - Let the fonts do the talking
4. **Performance** - Fast and lightweight

## Outcome

A clean, fast, and maintainable portfolio website that puts content first.
